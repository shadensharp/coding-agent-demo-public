# Demo Repo

This repository is the fixed Python workspace used by the coding-agent demo.

Current contents:
- `todo_api.py`: small in-memory Todo helpers
- `tests/test_todo_api.py`: baseline unittest coverage

Intentional V1 gaps for later demo work:
- list ordering does not use `priority` yet
- `apply_patch_update()` does not preserve unspecified fields in partial updates
